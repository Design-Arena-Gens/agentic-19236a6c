Intel Core i5-4570

Processeur 4ème génération (Haswell)

⚠️ Important : RAM vs Processeur

Le processeur Intel Core i5-4570 n'a pas de RAM intégrée. La RAM (mémoire vive) est un composant séparé de votre ordinateur. Le processeur et la RAM sont deux composants différents qui travaillent ensemble.

Note : La quantité de RAM installée dans votre ordinateur dépend de votre configuration. Ce processeur supporte jusqu'à 32 GB de RAM DDR3-1600, mais la quantité réelle installée peut varier (généralement 4 GB, 8 GB, 16 GB ou 32 GB selon votre système).

Comment vérifier votre RAM actuelle ?

Sur Windows :

  • Appuyez sur Windows + Pause
  • Ou allez dans Paramètres → Système → Informations système
  • La RAM installée est affichée

Via le Gestionnaire des tâches :

  • Appuyez sur Ctrl + Shift + Esc
  • Allez dans l'onglet Performance
  • Cliquez sur Mémoire
